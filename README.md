# 📦 쿠팡 고객상담관리센터 (Training Version)

**쿠팡 고객상담관리센터**는 고객센터 상담원이 고객 정보, 주문 내역, 상담 이력을 효율적으로 조회하고 관리하기 위해 사용하는 **사내 상담 업무 전용 웹 시스템**입니다.

## 🚀 실행 환경
* **How to Run**: 
    * **최신 웹 브라우저(Chrome 권장)** - 별도 서버 및 추가 설치 없이 실행 가능 합니다.
* **Development Style**: 
    * **Web Standards** - `<header>`, `<nav>`, `<main>`, `<section>` 등 시맨틱 태그를 준수한 마크업 구성
    * **Modern JS** - 유지보수와 동적 데이터 처리를 위해 `let` 키워드와 화살표 함수를 활용한 로직 구현

## 🛠 실행 방법
1. 프로젝트 폴더 내의 `index.html` 파일을 브라우저(Chrome, Edge 등)로 실행합니다.
2. 별도의 서버 설정 없이 로컬 환경에서 즉시 구동이 가능합니다.
3. 상단 네비게이션 탭을 클릭하여 각 메뉴로 자유롭게 이동할 수 있습니다.

## ✨ 주요 기능
### 1. 실시간 상담 현황 대시보드
* 오늘 접수된 문의, 처리 대기, 처리 중, 완료 건수를 카드 UI로 시각화하여 제공합니다.
* 시스템 점검 등 상담원이 숙지해야 할 주요 공지사항을 상단에 노출합니다.

### 2. 스마트 문의 목록 조회 및 검색
* `dummyTickets` 데이터를 활용하여 전체 문의 리스트를 테이블 형태로 렌더링합니다.
* **검색 기능**: 고객명, 주문번호, 문의 유형 키워드를 통해 특정 상담 내역을 실시간으로 필터링할 수 있습니다.

### 3. 문의 상세 내역 및 상담 처리
* **정보 조회**: 선택된 고객의 연락처, 주문 상품, 배송 상태를 한 화면에서 확인합니다.
* **상담 관리**: 상담 답변을 입력하고 처리 상태(`WAIT`, `PROGRESS`, `DONE`)를 변경하여 저장할 수 있습니다.

### 4. 사내 FAQ 관리
* `details` 및 `summary` 태그를 사용하여 자주 묻는 질문(로그인, 결제 오류, 배송 지연 등)을 아코디언 형태로 효율적으로 관리합니다.

## 🔄 기본 사용 흐름
1. **대시보드**: 전체적인 상담 현황과 공지사항을 확인합니다.
2. **문의 목록**: 문의 리스트에서 처리해야 할 업무를 선택하거나 검색합니다.
3. **상세 처리**: 상세 페이지에서 고객 문의 내용을 검토하고 답변을 작성합니다.
4. **상태 업데이트**: 답변 작성 후 하단 셀렉트 박스를 통해 처리 상태를 갱신합니다.

## ⚠️ 이상 상황 안내
* **답변 누락**: 상담 답변 내용이 비어있는 상태에서 저장 시, `aria-live` 영역을 통해 입력 요청 메시지가 표시됩니다.
* **검색 결과 없음**: 키워드와 일치하는 데이터가 없을 경우 리스트가 초기화되어 표시됩니다.
* **반응형 대응**: 화면 너비가 768px 이하로 줄어들 경우, 모바일 기기에 최적화된 1단 레이아웃으로 자동 전환됩니다.
